# Contributor Covenant Code of Conduct

This project follows the Contributor Covenant v2.1. We expect everyone to be
respectful, kind, and collaborative. Be considerate in all interactions and
follow established communication norms.

If you witness or experience unacceptable behavior, please contact the
repository owner: aronbisrat (https://github.com/aronbisrat)

Full text: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
